* Overview

This repository contains my personal [[https://github.com/doomemacs/doomemacs][Doom Emacs]] configuration. Doom Emacs is a configuration framework for GNU Emacs that provides a curated experience with sensible defaults, evil mode, and a modular approach to configuration.

* Installation

** Prerequisites

- GNU Emacs 27.1 or higher
- Git 2.23 or higher
- [[https://github.com/doomemacs/doomemacs][Doom Emacs]] installed

** Setup

Clone this repository to your Doom Emacs configuration directory:

#+begin_src sh
git clone https://github.com/KyoukoMejiro/doom-emacs/ ~/.config/doom
#+end_src

Then synchronize your Doom Emacs installation:

#+begin_src sh
doom sync
#+end_src

Restart Emacs or reload your configuration with:

#+begin_src emacs-lisp
M-x doom/reload
#+end_src

* Maintenance

** Updating Doom

To update Doom Emacs and packages:

#+begin_src sh

#+end_src

** Syncing Configuration

After making changes to configuration files:

#+begin_src sh
doom sync
#+end_src

** Cleanup

To remove orphaned packages:

#+begin_src sh
doom sync -p
#+end_src
